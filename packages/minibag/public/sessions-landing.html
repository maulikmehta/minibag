Everything You Need

Built-in features for multi-participant coordination

Multi-Participant Sessions

Solo or group modes with configurable limits. Support 3, 20, or unlimited participants per session.

Real-Time Updates

Built-in Socket.io integration. Changes sync instantly across all participants.

Privacy by Design

No phone numbers required. First names and nicknames only. Full control over data sharing.

Infinitely Configurable

Product tiers, custom limits, feature flags. Configure per-session or per-product.

Production Ready

Built-in validation, error handling, rate limiting. PostgreSQL + Supabase ready.

WhatsApp Integration

Share sessions instantly via WhatsApp with formatted messages and deep links.

How It Works

Three simple steps to add group coordination to your app

1

Install & Configure

npm install @localloops/sessions

const config = {
  tier: 'free',
  maxInvited: 20
};
2

Create Sessions

One API call to create coordinated sessions

const session = await
  createSession(config);
3

Share & Sync

Real-time updates across all participants

await inviteParticipants(
